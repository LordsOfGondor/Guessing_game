# Guessing_game
This is a simple python guessing game.
This is a simple Python number guessing game.
The computer randomly chooses a number between 1 and 50, and the player tries to guess it.
The game continues until the correct number is guessed.

How the Game Works:

  The program picks a random number between 1 and 50

  The player enters their name

  The player keeps guessing numbers

  The program tells the player if the guess is:

  too low

  too high

  When the correct number is guessed:

  a congratulatory message is shown

  the number of tries is displayed

Rules:

  You must enter a number between 1 and 50

  The game will not stop until the correct number is guessed
